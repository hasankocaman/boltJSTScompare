🎯 Cypress Nedir?

Cypress, modern web uygulamaları için geliştirilmiş bir end-to-end (E2E) test framework'üdür. JavaScript/TypeScript ile yazılır ve gerçek tarayıcıda çalışır.

✨ Cypress'in Avantajları

  • Kolay kurulum ve kullanım
  • Gerçek zamanlı test çalıştırma
  • Otomatik bekleme (automatic waiting)
  • Güçlü debugging araçları
  • Zaman yolculuğu (time travel) debugging
  • Screenshot ve video kayıt

⚙️ Kurulum

1Node.js Kurulumu

Cypress Node.js gerektirir. Node.js resmi sitesinden indirin ve kurun.

Terminal - Versiyon Kontrolü
node --version
npm --version
2Yeni Proje Oluşturma
Terminal - Proje Kurulumu
mkdir cypress-test-project
cd cypress-test-project
npm init -y
3Cypress Kurulumu
Terminal - Cypress Yükleme
npm install --save-dev cypress
# Veya yarn kullanıyorsanız
yarn add --dev cypress

🎯 Sonuç ve Öneriler

🚀 Hızlı Başlangıç İpuçları

  • Küçük testlerle başlayın ve yavaş yavaş kompleksleştirin
  • Data attributes kullanarak element seçimini optimize edin
  • Custom commands ile tekrarlayan işlemleri otomatikleştirin
  • API testlerini UI testleri ile kombine ederek daha kapsamlı senaryolar oluşturun
  • Fixture dosyaları ile test verilerinizi organize edin
  • Page Object Pattern ile test kodunuzu maintainable hale getirin
